QUICK SETUP GUIDE
==================

1. INSTALL DEPENDENCIES
   pip install -r requirements.txt

2. SET UP GOOGLE SHEETS API
   a. Go to https://console.cloud.google.com/
   b. Create/select a project
   c. Enable "Google Sheets API"
   d. Create Service Account credentials
   e. Download JSON key file
   f. Rename to "credentials.json"
   g. Place in project root

3. SHARE GOOGLE SHEET
   a. Open credentials.json
   b. Find "client_email" value
   c. Open your Google Sheet
   d. Click "Share"
   e. Add the service account email with "Viewer" permission

4. RUN DASHBOARD
   streamlit run app.py

That's it! The dashboard will open in your browser.
